# RESTful API for User Management using Laravel Sanctum

## EndPoint:

| Method | Endpoint  |           Used To           |
| :----: | :-------: | :-------------------------: |
|  Post  | /register |       Create new user       |
|  Post  |  /login   |        Create token         |
|  Get   | /profile  | Fetch user data using token |
|  Get   |  /logout  |        Delete token         |
